# 🍽️ Tasty Talks – A Social Platform for Food Enthusiasts

Tasty Talks is a full-stack social media platform designed for food lovers, home cooks, chefs, and culinary enthusiasts. The application allows users to discover, share, and interact with recipes while building a food community.

## 🚀 Features

- JWT-based Login and Registration  
- Three Roles: FOODIE, CHEF, ADMIN  
- Recipe Posting, Editing, and Management  
- Likes, Comments, Ratings, and Bookmarks  
- Follow / Unfollow Users  
- Community Chat System  
- Leaderboard and Points System  
- Admin Dashboard and Recipe Approval  

---

## 💡 Use Cases

- Share and discover recipes  
- Build a community of food enthusiasts  
- Help chefs gain followers and recognition  
- Enable interaction through comments and chat  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Spring Boot, Spring Security, JWT  
- **Database:** MySQL  
- **Real-Time Features:** WebSocket  
- **Deployment:** Render  

---

## 📌 Conclusion

Tasty Talks combines recipe sharing, social interaction, and community features into a single modern web application for food enthusiasts.
