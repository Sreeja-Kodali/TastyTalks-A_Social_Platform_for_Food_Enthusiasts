# Tasty Talks – A Social Platform for Food Enthusiasts

Tasty Talks is a full-stack social media platform designed for food lovers, home cooks, chefs, and culinary enthusiasts. The application allows users to discover, share, and interact with recipes while building a community around cooking and food.

# 🚀 Features

### Authentication & User Roles

* Secure user registration and login using JWT authentication.
* Three different user roles:

  * FOODIE
  * CHEF
  * ADMIN
* Role-based access control for different pages and features.

### Recipe Sharing

* Chefs can create, edit, and delete recipes.
* Recipes include:

  * Title
  * Description
  * Cuisine
  * Category
  * Difficulty
  * Ingredients
  * Instructions
  * Image URL
  * Tags
* Recipes can be saved with a pending approval status.

### Recipe Interaction

* Users can:

  * Like recipes
  * Bookmark recipes
  * Rate recipes from 1–5 stars
  * Add comments to recipes
* Trending and featured recipes are displayed on the home page.

### Profile System

* Personalized profile page for every user.
* Displays:

  * Avatar
  * Followers and Following
  * Bookmarks
  * Posted Recipes
  * Weekly Points
  * Total Points
* Separate dashboard layouts for FOODIE and CHEF users.

### Community Features

* Community chat system for interaction between users.
* Follow and unfollow chefs or other users.
* Leaderboard displaying top chefs based on points and followers.
* Notification system for activity updates.

### Admin Panel

* Admin can:

  * Approve or reject recipe submissions
  * Manage users and roles
  * Delete inappropriate comments
  * Feature recipes on the homepage
  * View platform statistics

# 💡 Use Cases

* Recipe Sharing Community – Users can share their own recipes and discover new dishes.
* Chef Promotion – Helps chefs gain followers and build an audience.
* Food Inspiration – Provides recipe ideas based on cuisine, category, and popularity.
* Social Interaction – Encourages discussion and engagement among food enthusiasts.
* Cooking Challenges – Can be extended with contests and chef rankings.
